What this file contains

This single HTML file demonstrates usage of semantic elements, text-level semantics, forms, tables, lists, media, SVG, Canvas, microdata, ARIA, and many useful attributes such as data-*, hidden, contenteditable, and more.

Note: It is illustrative — a few deprecated tags (like <font>) are intentionally omitted. It is also annotated to help learning.
